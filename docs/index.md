# PROTEUS Visual Language — brand guide

The written brand guide. For the interactive version see the
[Pages site](https://formingworlds.github.io/proteus-visual-language)
(source in [`../site/`](../site/)); a PDF is at
[brand-guide.pdf](https://formingworlds.github.io/proteus-visual-language/brand-guide.pdf).

Thermocline encodes one story: a rocky planet's transition from a **magma
ocean** to a **water world**. Red = hot state, black = deep time, blue = cool
state. Dark-first, instrument-grade, editorial.

## The five rules

1. **One story:** every surface reads hot → cold, inside → out. Never reverse
   the phase gradient.
2. **Dark-first:** Void `#05070B` is the default surface; Paper `#F2F5F7` is the
   mirror. Never pure black/white.
3. **Magma is scarce:** one hot moment per surface. Blue (Ocean/Azure) is the
   interactive/living layer.
4. **Type does the work:** tight Sora display, Instrument Sans body, Spline Sans
   Mono for every label and number.
5. **Homogeneity over novelty:** a docs page and a poster must read as one
   product. Everything traces to [`../tokens/tokens.css`](../tokens/tokens.css).
