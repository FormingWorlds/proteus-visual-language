05 · CONSTRUCTION — rules for every reproduction
SPEC
GLYPH
Invariant. The ring geometry is never redrawn, rotated, mirrored, or reproportioned. Color is the only variable.
GRADIENT
Radial, centered on the core: Magma #E23D28 core → Mantle #8E1F12 → Ocean #1B6FA8 → Ice #A8D4E8 at the outer arcs. Hot inside, cool outside — the planet cooling, read outward.
WORDMARK
Sora ExtraBold (800), all caps, letter-spacing +1%. The glyph replaces the O, set at 1.06× the type size so the outer arcs overshoot the cap height — as in the original mark.
TAG
"framework" in Spline Sans Mono Medium, lowercase, tracked +52%, right-aligned and tucked into the wordmark’s lower-right corner: raised 0.03× and inset 0.32× of the wordmark size (box measures; the tracking trails the last letter, whose ink lands ≈0.41× in). Drop it below 200px lockup width.
CLEARSPACE
1× cap height on all sides, minimum.
MINIMUM SIZE
Lockup ≥ 90px wide on screen. Below that: glyph + side-set name, or glyph alone.
DON'T
  • Don't set the wordmark in any face but Sora, or in mixed case.
  • Don't recolor letters — they are Paper #F2F5F7 on dark, Ink #10151B on light. The glyph carries all color.
  • Don't reverse the gradient (cool core / hot rim reads as the wrong physics).
  • Don't apply glows, shadows, or outlines to the mark.
  • Don't place the phase glyph on mid-tone or busy surfaces — use a mono colorway there.
  • Don't rebuild the glyph from CSS circles or SVG arcs — always reproduce from the master asset.
THE STORY IN ONE MARK
A molten core cooling through deep time into an ocean world — PROTEUS's entire scientific narrative, read from the center of the glyph outward. Red → black → blue is also the data-viz scale, so every figure quietly repeats the logo.
